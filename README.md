# 부모님 자유여행용 나트랑 3박 5일 일정표

> 50~60대 부모님 기준으로 **무리 없는 동선**, **충분한 휴식**, **초보자도 바로 따라할 수 있는 구성**으로 정리했습니다.

## 여행 전 기본 정보 (핵심 요약)

- **여행 형태**: 3박 5일 (보통 밤 비행기 출발/도착)
- **이동 원칙**: 하루 2~3개 핵심 장소만 방문 + 카페/호텔 휴식 시간 확보
- **교통 원칙**: 도보 최소화, `그랩(Grab)` 중심 이동
- **식사 원칙**: 너무 먼 맛집보다 호텔/관광지 근처 평점 좋은 곳 우선

## 3박 5일 일정표 (오전 / 점심 / 오후 / 저녁)

| 구분 | 일정 | 부모님 여행 팁 |
|---|---|---|
| 오전 | 출국 준비, 공항 이동, 체크인 | 여권/항공권/유심핀/상비약 미리 한 파우치에 정리 |
| 점심 | 기내식 또는 공항 간단 식사 | 기름진 음식 피하고 수분 충분히 섭취 |
| 오후 | (비행) 나트랑 깜란공항(CXR) 도착 → 호텔 이동 | 공항에서 바로 환전 소액만 진행 후 시내에서 추가 환전 |
| 저녁 | 호텔 체크인 후 주변에서 가벼운 식사, 조기 취침 | 첫날은 무조건 휴식 중심(마사지 60분 정도는 가능) |

| 점심 | 사원 근처 또는 시내 식당에서 쌀국수/분짜 | 에어컨 있는 매장 우선 선택 |

## 공항 이동 / 환전 / 유심 / 그랩 사용 팁

### 1) 공항 이동 팁
- **깜란공항(CXR) → 나트랑 시내**: 차량으로 보통 40~60분
- 밤 도착 시 피로도가 높으므로, **사전 픽업 예약** 또는 **그랩 호출** 추천
- 출국일은 최소 **비행 3시간 전 공항 도착** 기준으로 역산

### 2) 환전 팁
- 공항에서는 **필요한 만큼만 소액 환전**
- 큰 금액은 시내 환전소(금은방/환전소)에서 환율 비교 후 환전
- 소액권(예: 50,000동/100,000동) 위주로 준비하면 결제 편리

### 3) 유심(eSIM/물리심) 팁
- 초보자는 **한국에서 eSIM 사전 구매**가 가장 편함
- 물리심은 공항/시내 구매 가능하나 개통 대기시간 발생 가능
- 데이터는 지도+그랩+메신저 기준 **하루 1~2GB**면 충분

### 4) 그랩(Grab) 사용 팁
- 앱 설치/카드 등록은 한국에서 미리 완료
- 호출 시 차량번호 확인 후 탑승 (기사 이름도 함께 체크)
- 번화가/호텔 앞은 픽업 포인트를 정확히 지정
- 기사와 소통 어려우면 **호텔명/목적지 주소를 앱 채팅으로 전송**
